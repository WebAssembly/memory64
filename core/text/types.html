Note

The optional identifier names for parameters in a function type only have documentation purpose. They cannot be referenced from anywhere.

\[\begin{split}\begin{array}{llclll} \def\mathdef4534#1{{}}\mathdef4534{recursive type} & \href{../text/types.html#text-typedef}{\mathtt{typedef}} &\equiv& \def\mathdef4722#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4722{(}~~\def\mathdef4723#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4723{rec}~~\href{../text/types.html#text-typedef}{\mathtt{typedef}}~~\def\mathdef4724#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4724{)} \\ \end{array}\end{split}\]

Similarly, final sub types with no super-types can omit the \(\href{../syntax/types.html#syntax-subtype}{\mathsf{sub}}\) keyword and arguments:

\[\begin{split}\begin{array}{llclll} \def\mathdef4534#1{{}}\mathdef4534{sub type} & \href{../text/types.html#text-comptype}{\mathtt{comptype}} &\equiv& \def\mathdef4725#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4725{(}~~\def\mathdef4726#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4726{sub}~~\def\mathdef4727#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4727{final}~~\epsilon~~\href{../text/types.html#text-comptype}{\mathtt{comptype}}~~\def\mathdef4728#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4728{)} \\ \end{array}\end{split}\]

Abbreviations¶

\[\begin{array}{llclll} \def\mathdef4534#1{{}}\mathdef4534{address type} & \def\mathdef4731#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4731{} &\equiv& \def\mathdef4732#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4732{i32} \end{array}\]
