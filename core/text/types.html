Note

The optional identifier names for parameters in a function type only have documentation purpose. They cannot be referenced from anywhere.

\[\begin{split}\begin{array}{llclll} \def\mathdef4346#1{{}}\mathdef4346{recursive type} & \href{../text/types.html#text-typedef}{\mathtt{typedef}} &\equiv& \def\mathdef4533#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4533{(}~~\def\mathdef4534#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4534{rec}~~\href{../text/types.html#text-typedef}{\mathtt{typedef}}~~\def\mathdef4535#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4535{)} \\ \end{array}\end{split}\]

Similarly, final sub types with no super-types can omit the \(\href{../syntax/types.html#syntax-subtype}{\mathsf{sub}}\) keyword and arguments:

\[\begin{split}\begin{array}{llclll} \def\mathdef4346#1{{}}\mathdef4346{sub type} & \href{../text/types.html#text-comptype}{\mathtt{comptype}} &\equiv& \def\mathdef4536#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4536{(}~~\def\mathdef4537#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4537{sub}~~\def\mathdef4538#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4538{final}~~\epsilon~~\href{../text/types.html#text-comptype}{\mathtt{comptype}}~~\def\mathdef4539#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4539{)} \\ \end{array}\end{split}\]

Abbreviations¶

The index type can be omited, in which case it defaults \(\href{../syntax/types.html#syntax-valtype}{\mathsf{i32}}\):

\[\begin{array}{llclll} \def\mathdef4346#1{{}}\mathdef4346{index type} & \def\mathdef4542#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4542{} &\equiv& \def\mathdef4543#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4543{i32} \end{array}\]
